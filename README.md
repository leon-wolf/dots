**Setup**

+ OS                    : `Artix Linux`
+ Window Manager        : `2bwm`
+ Shell                 : `zsh`
+ Applicantion Launcher : `rofi`
+ Bar                   : `tint2`
+ Compositor            : `compton`
+ Notifications         : `dunst` / `libnotify`
+ Music Player          : `mps-youtube`
+ Video Player          : `mpv`
+ File Manager          : `ranger`
+ Image Viewer          : `sxiv`
+ Text Editor           : `neovim`
+ Terminal Emulator     : `urxvt`
+ Terminal Multiplexer  : `tmux`
+ Web Browser           : `firefox`
+ Download Manager      : `aria2c`
